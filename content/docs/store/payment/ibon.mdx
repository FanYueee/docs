---
title: 【台灣限定】711/萊爾富 超商代碼
description: 如果您在付款方式選擇 【台灣限定】711/萊爾富 超商代碼 以下將指引您如何繳款。
---

import { Step, Steps } from 'fumadocs-ui/components/steps';

<Steps>
<Step>
選擇 【台灣限定】711/萊爾富 超商代碼 進入帳單頁面，您將會看到以下圖片的繳款資訊。
![ibon1](/images/store/payment/ibon/ibon1.png)
</Step>
<Step>
接著請依照以下圖片指示至711或萊爾富機台列印繳費單，再將繳費單拿至櫃檯繳費。

如果是 711 Ibon 點擊左上代碼輸入後可以選擇掃描 QRCode，直接將帳單上的 QRCode 掃描即可免自行輸入繳費代碼。

![7-ELEVEN-ibon.jpg](/images/store/payment/ibon/7-ELEVEN-ibon.jpg)
</Step>
</Steps>

import { Accordion, Accordions } from "fumadocs-ui/components/accordion";

</Accordions>
