import { Step, Steps } from 'fumadocs-ui/components/steps';

<Steps>
<Step>
選擇 【台灣限定】全家超商代碼 進入帳單頁面，您將會看到以下圖片的繳款資訊。
![famiport1](/images/store/payment/famiport/famiport1.png)
</Step>
<Step>
接著請依照以下圖片指示至全家機台列印繳費單，再將繳費單拿至櫃檯繳費。
![FAMI-FAMIPORT.jpg](/images/store/payment/famiport/FAMI-FAMIPORT.jpg)
</Step>
</Steps>
