---
title: 【台灣除外】PayPal
description: 如果您在付款方式選擇 【台灣除外】PayPal 超商代碼 以下將指引您如何繳款。
---

import { Step, Steps } from 'fumadocs-ui/components/steps';

<Steps>
<Step>
選擇 【台灣除外】PayPal 超商代碼 進入帳單頁面，您將會看到以下圖片的繳款資訊。
![paypal1](/images/store/payment/paypal/paypal1.png)
</Step>
<Step>
接著點擊帳單上的「**立即付款**」按鈕後，會導向以下圖片之頁面。
![paypal2](/images/store/payment/paypal/paypal2.png)
</Step>
<Step>
接著點擊您可以選擇綁定 PayPal 帳號，或直接使用信用卡付款。

綁定將登入 PayPal 帳號，綁定完成點選下方「**提交付款**」按鈕即可。

import { Callout } from 'fumadocs-ui/components/callout';

<Callout title="注意" type="warn">
	綁定 PayPal 帳號或是輸入信用卡資訊，都不會自動續約付款，因此續約時依舊需要自行至客戶中心繳費！
</Callout>

</Step>
</Steps>

import { Accordion, Accordions } from "fumadocs-ui/components/accordion";

<Accordions>
	<Accordion title="超過繳費期限還能夠繳費嗎？">
        不能唷，超過後繳費資訊會自動失效，如果您需要繳費請聯繫客服人員處理。
	</Accordion>
</Accordions>
