---
title: 【台灣限定】四大超商條碼
description: 如果您在付款方式選擇 【台灣限定】四大超商條碼 以下將指引您如何繳款。
---

import { Step, Steps } from 'fumadocs-ui/components/steps';

<Steps>
<Step>
選擇 【台灣限定】四大超商條碼 進入帳單頁面，您將會看到以下圖片的繳款資訊。
![barcode1](/images/store/payment/barcode/barcode1.png)
</Step>
<Step>
點選「**顯示繳費條碼**」按鈕後將出現以下頁面
![barcode2](/images/store/payment/barcode/barcode2.png)
</Step>
<Step>
直接持畫面上的條碼給予四大超商店員掃描即可繳費。
</Step>
</Steps>

import { Accordion, Accordions } from "fumadocs-ui/components/accordion";

<Accordions>
	<Accordion title="我繳完費了，但為什麼等了很久遲遲沒有顯示已付款？">
		四大超商代碼因超商金流核銷需要 2~3 天的時間。
	</Accordion>
	<Accordion title="超過繳費期限還能夠繳費嗎？">
        不能唷，超過後繳費資訊會自動失效，如果您需要繳費請聯繫客服人員處理。
	</Accordion>
</Accordions>
