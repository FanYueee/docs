---
title: 升級服務
description: 目前資源使用上不夠嗎？那可以透過客戶中心上的升級服務，來完成服務上資源的升級。
---

import { Step, Steps } from 'fumadocs-ui/components/steps';
import { Callout } from 'fumadocs-ui/components/callout';

<Callout title="注意" type="warn">
    申請取消服務即表示您知道並同意取消服務的相關免責聲明。
</Callout>


<Steps>
<Step>
### 登入客戶中心
登入[客戶中心](https://client.vproxy.cloud)後，點選上方選單中的「**服務**」下的「**我的服務**」
![upgrade1](/images/store/services/upgrade/upgrade1.png)
<Callout title="注意" type="info">
    需要先登入客戶中心，才會看到選單中有服務選項。
</Callout>
</Step>
<Step>
### 點選欲升級的服務
本篇以下圖橘色框服務為例，直接點下去。
![upgrade1](/images/store/services/upgrade/upgrade2.png)
</Step>
<Step>
### 點選要求撤銷
進入後，點選左側「**動作**」下的「**升級/降級 選項**」。
![upgrade3](/images/store/services/upgrade/upgrade3.png)
</Step>
<Step>
### 選擇升級的資源量
以本篇為例，將從 10GB 記憶體升級為 12GB。

選擇完成後點擊下方的「**繼續**」按鈕。
![upgrade4](/images/store/services/upgrade/upgrade4.png)

</Step>
<Step>
1. 橘色框的部分是輸入優惠代碼。
2. 藍色框的部分選擇您要的付款方式，如果是全額使用餘額付款則無須理會。
3. 選擇和輸入完畢後點擊下方綠色框內的「**繼續**」按鈕
![upgrade5](/images/store/services/upgrade/upgrade5.png)
</Step>
<Step>
### 帳單生成
現在依照帳單上資訊繳費就完成了！
![upgrade6](/images/store/services/upgrade/upgrade6.png)
</Step>
</Steps>
