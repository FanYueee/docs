---
title: 設定帳號
description: 本篇將引導您第一次購買服務後設定控制面板帳號及密碼。
---

import { Step, Steps } from 'fumadocs-ui/components/steps';

<Steps>
<Step>
當您第一次購買與繳費完成後，稍待一段時間確認完畢後，您將可以在您的「電子信箱」收取到以下此封信件。
![account1.png](/images/panel/account2/account1.png)
</Step>
<Step>
點選信件上的「**設定您的密碼**」後，將會跳轉至以下頁面。

接著請輸入您欲設置的密碼。
![account2.png](/images/panel/account2/account2.png)
</Step>
<Step>
接著您就可以進入控制面板操作伺服器了！
</Step>
</Steps>
