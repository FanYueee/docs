---
title: 重設密碼
description: 如果您忘記了您的控制面板帳號與密碼，別擔心您可以透過以下指引進行重設密碼。
---

import { Step, Steps } from 'fumadocs-ui/components/steps';

<Steps>
<Step>
進入[伺服器控制面板](https://panel.vproxy.cloud)，進入後點擊下方的「**忘記密碼**」。
![password1](/images/panel/password/password1.png)
</Step>
<Step>
進入後填入您帳號的 Email 後，點擊下方的「**發送 E-MAIL**」。
![password2](/images/panel/password/password2.png)
</Step>
<Step>
現在進入您的電子郵箱找到以下郵件，找到後點擊「**重置密碼**」。
![password3](/images/panel/password/password3.png)
</Step>
<Step>
進入後即可輸入您要更改為的密碼。
![password4](/images/panel/password/password4.png)
</Step>
<Step>
最後點擊「**重設密碼**」即完成了！
</Step>
</Steps>
