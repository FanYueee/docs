什麼是 SFTP？SFTP 全名是 SSH File Transfer Protocol（Secret File Transfer Protocol）這是一種檔案傳輸協定，可為檔案傳輸提供安全加密方式。他將要求用戶要跟伺服器進行驗證，且資料傳輸時會透過 SSH 安全通道傳輸。

SFTP 可用於上傳或下載資料夾、大型檔案。

> 本篇以 [FileZilla](https://filezilla-project.org) 為例，介紹如何使用 SFTP 傳輸檔案。

## 取得 SFTP 連線資訊

<Steps>
<Step>
進入伺服器控制面板，在左側選單找到「**其他設定**」的「**SFTP 資訊**」。
![sftp1](/images/panel/server/sftp/1.png)
</Step>
<Step>
打開 FileZilla 將控制面板上的「**SFTP 連線位置**」複製到「**主機**」位置，「**使用者名稱**」複製到「**使用者名稱**」，「**密碼**」與控制面板的密碼是一致的，請輸入控制面板密碼。
![sftp2](/images/panel/server/sftp/2.png)
如果忘記密碼可依照此教學進行重新設置：[重設密碼](https://docs.vproxy.cloud/docs/panel/password)
![sftp3](/images/panel/server/sftp/3.png)
</Step>
<Step>
點擊「**快速連線**」按鈕，FileZilla 將會連線到您的伺服器。
![sftp4](/images/panel/server/sftp/4.png)
</Step>
<Step>
如果出現此訊息「**未知的主機金鑰**」直接點擊「**確定**」即可。
也可將「總是信任此主機」勾選，再點擊「確定」，下次連線時不會再出現此訊息。
![sftp5](/images/panel/server/sftp/5.png)
</Step>
<Step>
如果右側面板顯示檔案結構，表示連線成功。
![sftp6](/images/panel/server/sftp/6.png)
</Step>
<Step>
現在可以使用拖曳方式將檔案從左側（本機）拖到右側（伺服器），也可以直接拉一個資料夾視窗裡面拖曳檔案到右側面板進行上傳。
![sftp7](/images/panel/server/sftp/7.gif)
![sftp8](/images/panel/server/sftp/8.gif)
</Step>  
<Step>
下面的區域會顯示目前正在上傳或下載的檔案狀態，您可以在這裡查看傳輸進度。以及是否傳輸成功。
![sftp9](/images/panel/server/sftp/9.png)
</Step>
</Steps>

<Callout type="info">
如果要刪除大量資料，建議使用面板的檔案管理進行刪除，因為 SFTP 會一個一個刪除檔案，速度會比較慢。
</Callout>
<Callout type="info">
    如果上傳的為壓縮檔案，可以依照此教學進行解壓縮：[解壓縮檔案](https://docs.vproxy.cloud/docs/panel/server/files#%E8%A7%A3%E5%A3%93%E7%B8%AE%E6%AA%94%E6%A1%88)

    解壓縮僅支援 `.zip` `.tar` `.gz` `.tar.gz` 檔案，並**不支援** `.rar`，另外如果壓縮檔案內有中文名稱的檔案將會導致解壓縮失敗，請先將名稱改掉，或開啟客服單請技術人員由後台處理。
</Callout>
