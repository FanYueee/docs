---
title: 備份管理
description: 備份管理提供伺服器的備份功能，並且不佔用伺服器本身的硬碟使用量。
---

import { Step, Steps } from 'fumadocs-ui/components/steps';

<Steps>
<Step>
進入備份管理後，點擊右側的「**創建備份**」按鈕。
![backups1](/images/panel/server/backups/backups1.png)
</Step>
<Step>
您可以輸入方便辨識的備份名稱，完成後點擊下方的「**開始備份**」。
![backups2](/images/panel/server/backups/backups2.png)
</Step>
<Step>
等待備份完成（時間將取決於伺服器檔案大小）。
![backups3](/images/panel/server/backups/backups3.png)
</Step>
<Step>
備份完成右側將會出現三個點。
![backups4](/images/panel/server/backups/backups4.png)
</Step>
<Step>
點擊三個點後，即可對備份進行下載／刪除／鎖定或還原。
![backups5](/images/panel/server/backups/backups5.png)
</Step>
</Steps>
