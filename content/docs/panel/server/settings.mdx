import { Callout } from 'fumadocs-ui/components/callout';

以下設定可以在伺服器控制面板上排選單找到「**其他設定**」。
![settings1](/images/panel/server/settings/settings1.png)

## SFTP 連線資訊
您可以透過 SFTP 進行傳輸檔案，可以使用 [FileZilla](https://filezilla-project.org) 或 [WinSCP](https://winscp.net/eng/index.php) 擇一軟體連線。

FileZilla 安裝教學：[https://mcs.wiki/software/filezilla/](https://mcs.wiki/software/filezilla/)

如果不知道如何使用，可以參考我們的操作指引[SFTP 操作指引](https://docs.vproxy.cloud/panel/server/sftp)
![settings2](/images/panel/server/settings/settings2.png)
## 更改伺服器資訊
在這裡您可以更改伺服器在面板顯示的名稱和備註，或許可以幫助您更好分辨伺服器！
![settings3](/images/panel/server/settings/settings3.png)
## 維修資訊
維修資訊欄位是如果您需要向客服提出協助，請務必提供以下的「伺服器唯一辨識碼」給予客服人員。
![settings4](/images/panel/server/settings/settings4.png)
## 重新安裝伺服器
<Callout title="注意" type="warn">
    重新安裝可能會導致資料被覆蓋，因此重新安裝前，建議先備份原有的資料，以免資料找不回來。
</Callout>
如果您需要重新安裝伺服器的核心，可透過以下重新安裝功能處理，如果您需要更換核心版本請至「啟動設定」，如果您是要將伺服器檔案恢復預設，請先將所有資料刪除或移至其他資料夾，再點選重新安裝。
![settings5](/images/panel/server/settings/settings5.png)
