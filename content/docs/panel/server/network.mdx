## 創建端口
<Steps>
<Step>
進入網路設定後，點選「**創建端口**」按鈕。
![network1](/images/panel/server/network/network1.png)
</Step>
<Step>
現在您的伺服器有兩個端口可用了！
![network2](/images/panel/server/network/network2.png)
</Step>
</Steps>
## 更改主要端口
如果想更換主要端口（玩家連線的端口），可點選希望切換成的端口，然後點擊「**切換主要**」。
![network3](/images/panel/server/network/network3.png)
