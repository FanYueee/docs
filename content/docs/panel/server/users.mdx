---
title: 共同用戶
description: 共同用的功能，可以讓您的伺服器控制面板部分或所有的權限分享給其他人一起操作控制。
---

import { Step, Steps } from 'fumadocs-ui/components/steps';
import { Callout } from 'fumadocs-ui/components/callout';

<Steps>
<Step>
進入「**共同用戶**」，接著點擊「**創建新共同用戶**」按鈕。
![users1](/images/panel/server/users/users1.png)
</Step>
<Step>
1. 於黃色框區域輸入欲邀請一同控制伺服器的管理者「**電子郵箱（E-Mail）**」。
2. 於藍色框區域選擇欲分享哪些控制權限。
3. 選擇完成後點擊右上的「**邀請共同用戶**」

![users2](/images/panel/server/users/users2.png)
</Step>
<Step>
接著你邀請的共同用戶，他應該會在電子郵箱收到以下信件。
![users3](/images/panel/server/users/users3.png)
<Callout title="注意" type="info">
    如果該共同用戶未使用過 vProxy Cloud，還應收到以下信件，請先點擊此信件的「**設定您的密碼**」進行設置後在點擊「**訪問伺服器**」按鈕。
</Callout>
![users4](/images/panel/server/users/users4.png)
</Step>
</Steps>
