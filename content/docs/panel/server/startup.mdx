以下設定可以在伺服器控制面板上排選單找到「**啟動設定**」。
![startup1](/images/panel/server/startup/1.png)

## 啟動參數
啟動參數為當前伺服器的啟動參數。
![startup2](/images/panel/server/startup/2.png)
## 啟動版本
啟動版本可以調整啟動的版本，Minecraft 伺服器為 Java 版本，Discord 機器人是對應語言的版本。
![startup3](/images/panel/server/startup/3.png)
## 系統變數
在此可以變更執行的系統相關細節設定，部分設定需要重新安裝才會生效（如伺服器版本）。
![startup4](/images/panel/server/startup/4.png)
