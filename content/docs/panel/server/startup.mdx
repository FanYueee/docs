以下設定可以在伺服器控制面板上排選單找到「**啟動設定**」。
![startup1](/images/panel/server/startup/startup1.png)

## 啟動參數
啟動參數為當前伺服器的啟動參數。
![startup2](/images/panel/server/startup/startup2.png)
## 啟動版本
啟動版本可以調整啟動的版本，Minecraft 伺服器為 Java 版本，Discord 機器人是對應語言的版本。
![startup3](/images/panel/server/startup/startup3.png)
## 系統變數
在此可以變更執行的系統相關細節設定，部分設定需要重新安裝才會生效（如伺服器版本）。
![startup4](/images/panel/server/startup/startup4.png)
## 更換核心類型
在此您可以更換核心的類型，像是 Fabric、Forge、Paper 都能夠在此更換。
<Steps>
<Step>
點擊「**更換核心類型**」
![startup5](/images/panel/server/startup/startup5.png)
</Step>
<Step>
找到欲更換的類型後點擊「**安裝**」按鈕。
![startup6](/images/panel/server/startup/startup6.png)
</Step>
<Step>
點擊「**確定**」
![startup7](/images/panel/server/startup/startup7.png)
</Step>
<Step>
回到上一頁或是「**啟動設定**」

可以看到啟動參數的部分有變動，現在輸入您要安裝的版本資訊進去，如果是最新版本通常直接空著即可。
![startup8](/images/panel/server/startup/startup8.png)
</Step>
<Step>
點擊更改伺服器核心類型的「**重新安裝伺服器**」
<Callout title="注意" type="warn">
    重新安裝可能會導致資料被覆蓋，因此重新安裝前，建議先備份原有的資料，以免資料找不回來。
</Callout>
![startup9](/images/panel/server/startup/startup9.png)
</Step>
<Step>
現在等待伺服器進行重新安裝。
![startup10](/images/panel/server/startup/startup10.png)
</Step>
<Step>
最後伺服器可用時就代表安裝完畢！現在可以啟動伺服器確認了！
<Callout title="注意" type="info">
    如果點擊重新安裝超過 5 分鐘後伺服器還是無法使用，請洽詢客服人員協助處理。
</Callout>
</Step>
</Steps>
